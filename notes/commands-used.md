
# Phase 1: Server Baseline Assessment

## Task 1: Inspect the Linux Server

### Scenario

You have received access to a new Linux server.

Before making any changes, collect information about the environment.

### Objectives

Identify:

| Command              | Purpose                                                    |
| -------------------- | ---------------------------------------------------------- |
| `uname -a`              | Shows free and used space for entire mounted filesystems.  |
| `du -sh <directory>` | Shows the disk space used by a specific directory or file. |


 `uname -a` 
- Linux distribution
- Kernel version
  
  `whoami` 
- Current user

  `htop`
- CPU information
- Memory usage
- Disk usage
- Running processes
  
  `sudo ss -tulnp`
- Network information

### Deliverables

Document:

- Commands used
- System information collected
- Screenshots
- Observations
