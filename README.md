
### 1.Project Overview

This project simulates provisioning and administering a Linux server using common DevOps system administration tasks. It covers user management, permissions, storage management using LVM, service management with systemd, and troubleshooting.

### 2. Skills Demonstrated
- Linux Administration
- User & Group Management
- File Permissions
- Nginx Deployment
- LVM
- Systemd
- Log Analysis
- Troubleshooting
- Documentation
  
### 3. Environment
```
Ubuntu 24.04
Killercoda
Nginx
LVM
systemd
```

Linux VM

       ├── Users
